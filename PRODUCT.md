# Horror Kutty Satthan

## What It Is
A browser-based horror survival game. The player is stranded in a haunted forest, sheltered in a lone house, and must survive 100 consecutive nights against escalating ghost encounters. Strict win condition: outlast all 100 nights.

## Target Users
- Horror game fans who want a browser-accessible scare
- Casual to mid-core gamers drawn to defined challenge-run formats
- Players motivated by shareable achievements ("I survived all 100 nights")

## Core Game Loop
1. Day phase — Scavenge forest for supplies, repair damage, board windows, set traps, manage power/light
2. Night phase — Ghost AI activates; player repels or evades using barricades, light management, rationed resources
3. Escalation — Each new night raises threat: more ghosts, smarter patterns, new ghost types
4. Win/Lose — Survive night 100 = victory; caught by ghost = fast retry

## Key Features
- 100-night challenge arc with strict win condition
- Escalating Ghost AI (more ghosts, smarter, new types each phase)
- House fortification (board windows, barricade doors, manage electricity)
- Day-phase scavenging and resource management
- Atmospheric horror (dark visuals, fog, ambient sound, minimal UI)
- Score/streak tracking — "nights survived" as metric
- Fast retry loop

## Brand & Tone
Atmospheric horror — tense, slow-burn, genuinely unsettling. Not gory.
Platform: Web browser
